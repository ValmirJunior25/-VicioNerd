# VicioNerd
Plataforma para gamers gerenciarem sua coleção de jogos

## Tecnologias
- C# .NET 10
- SQLite com Entity Framework Core

## Estrutura
- Models/    → classes de dados
- Services/  → lógica de negócio  
- Data/      → configuração do banco
